# AI-Researcher Reference-Based Mode

Use this mode after building curated paper packs from Zotero and candidate papers.

## Input Packs

Preferred packs:

```text
01_welding_defect_yolo_core
02_yolo11_and_recent_yolo_improvements
03_small_object_detection
04_elongated_crack_defect_detection
05_localization_loss_functions
06_attention_neck_feature_fusion
07_transfer_validation_industrial_defects
```

## Prompt Template

```text
I have a curated pack of reference papers about [topic]. Please generate implementable ideas for improving YOLO11 on welding defect detection.

Constraints:
- Main problem: small, elongated, low-contrast, class-imbalanced welding defects.
- Core method must include or support morphology-aware localization loss and lightweight small-defect feature enhancement.
- Primary metric: mAP50-95.
- Second target: small/elongated class recall or AP.
- Efficiency must remain practical.
- Every proposed component must have a clear ablation.

Return ideas with paper-grounded motivation and mark which references support each component.
```

## Required Output

```text
idea_name
supporting_references
mechanism
why_it_fits_welding_defects
implementation_plan
ablation_matrix
expected_failure_cases
novelty_risk
```

## Formal Citation Rule

References generated by AI-Researcher are not formal citations until they are confirmed in the Zotero export.
